## Enunciado 

Exiba a arte abaixo.

```
 ||  || 
 \\()// 
//(__)\\
||    ||
```

_Observação: o caracter `\` tem uma função especial quando utilizado em uma string. Por enquanto, utilize `\\` quando quiser exibir `\`._
